🔹算数运算符

运算符 含义 示例 结果
+ 加法 x=1+1 x=2
- 减法 x=1-1 x=0
* 乘法 x=2*2 x=4
/ 除法 x=4/2 x=2
% 取余数 x=5%2 x=1

🔹赋值运算符

运算符 含义 示例 结果
= 直接赋值 x=1 x=1
+= x=x+ x+=1 x=2
-= x=x- x-=1 x=0
/= x=x/ x/=1 x=1
%= x=x% x%=1 x=0

🔹比较运算符

运算符 含义 示例 结果
== 等于 x==1 true
!= 不等于 x!=0 true
> 大于 x>1 false
< 小于 x<2 true
>= 大于等于 x>=1 true
<= 小于等于 x<=2 true
